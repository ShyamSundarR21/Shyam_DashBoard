# SmartKiosk AI — Retail Intelligence Platform

A full React + Tailwind CSS pitch deck / product UI for an AI-powered smart retail kiosk system.

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Headline, impact stats, and CTAs |
| **The Retail Crisis** | 4 key problem statistics |
| **Solutions** | 4 AI modules (Revenue, Vision, Persona, Magnet) |
| **Tech Architecture** | Input → Processing → Output → Reward pipeline |
| **Kiosk Demo** | Live Gen Z vs Senior persona switcher |

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- React 18 + Vite
- Tailwind CSS (dark futuristic theme)
- Syne + DM Sans + JetBrains Mono fonts
- Lucide React icons
- CSS animations (scan line, float, fade-up, pulse-glow)
